# Airport Travel Taxis (GCC Chauffeur Service)

Professional, scheduled, 24/7 cross-border and regional chauffeur services across the Gulf Cooperation Council (GCC) including Saudi Arabia, UAE, Kuwait, Qatar, and Bahrain.

## Key Features
- **Punctual & Professional**: High-standard chauffeur service for airport transfers, city-to-city, and international border crossings.
- **Premium Fleet**: Modern vehicles including Executive Sedans (Mercedes S-Class, BMW 7 Series), Large SUVs (GMC Yukon XL, Cadillac Escalade), and high-capacity vans for groups.
- **Secure & Reliable**: Fully pre-booked service with experienced drivers specialized in long-distance and border transit.
- **GCC-Wide Coverage**: Dedicated routes connecting major cities (Dubai, Riyadh, Doha, Kuwait City, Manama) and all regional international airports.

## Technical Details
- **Frontend**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Type Safety**: TypeScript

## Getting Started

### Prerequisites
- Node.js (Latest LTS)
- npm or yarn

### Installation
```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# Build for production
npm run build
```
